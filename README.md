# my_ip_phone

## 项目简介
基于Java实现的简单IP电话系统，使用TCP进行信令控制，UDP传输音频数据。

## 功能特性
- TCP连接：拨号、接听、挂断控制
- UDP传输：实时音频流
- 图形界面：Swing实现

## 技术实现

- **信令协议(TCP)**：DIAL、ACCEPT、HANGUP
- **音频格式**：8000Hz, 16bit, 单声道
- **网络端口**：TCP使用指定端口，UDP使用指定端口

## 使用说明

编译并运行项目中的 IPPhone.java 即可启动程序

### 本地测试

将项目拷贝为两个实例：

1. 启动第一个实例（监听方）
   - 地址和端口保持默认
   - 点击 "Listen"
2. 启动第二个实例（拨号方）
   - 地址和TCP端口保持默认
   - UDP端口填 6000
   - 点击 "Dial"
   - 连接成功后可进行语音通话
   - 点击 "Hangup" 结束通话

### 不同机器测试
监听方：点击 Listen，告知对方自己的IP地址

拨号方：输入对方IP地址，点击 Dial，连接成功后可进行语音通话
